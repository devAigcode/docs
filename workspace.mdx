## Personal Project Management

In the **Projects** panel of your Workspace, you can centrally manage all your project drafts and deployed applications. For a single project, you can perform the following actions:

<Frame>
  ![Image](/images/image-16.png)
</Frame>

<CardGroup cols={2}>
  <Card icon="pen" title="Edit">
    Click **Edit** to directly enter the AutoCoder.cc editor and continue modifying your project code or UI through natural language.
  </Card>
  <Card icon="eye" title="Preview">
    Click **Preview** to open a full-screen, immersive preview interface (consistent with the preview effect in the editor). If you need to make changes, you can click the quick access on the right to jump directly into edit mode.
  </Card>
  <Card icon="clone" title="Remix">
    Click the `...` menu in the top right corner of the project card and select **Remix**. The system will clone a brand new project draft based on the current project, making it convenient for destructive testing or derivative version development.
  </Card>
  <Card icon="trash" title="Delete">
    Select **Delete** in the `...` menu to permanently remove the project. _(Note: Please confirm that the project is unpublished or backed up before deleting)._
  </Card>
</CardGroup>

## Published Project Management

For projects that have already been published and are online, you can perform subsequent publication configuration and status control in the **Published** panel of the Workspace (or the Publish pop-up within the editor):

<Frame>
  ![Image](/images/image-17.png)
</Frame>

### Domain Settings

In the **Domain Setup** module, you can flexibly manage your project's access address:

- **Subdomain**: View or modify the system-assigned `.autocoder.cc` suffix domain.
- **Custom Domain**: Bind a domain you have purchased on a third-party platform to the current project. This requires configuring a CNAME and uploading an SSL certificate.

### Project Sharing and Public Visibility

By toggling the **Share to SparkVibe** switch, you can control whether the project is publicly displayed in the SparkVibe community. Once enabled, other users can like or Remix your work.

<Frame>
  ![Image](/images/image-20.png)
</Frame>

### External Link Redirection

After successful deployment, the system will provide a fixed **Website URL** (frontend link) and **Backend URL** (management backend link). Anyone (without needing an AutoCoder account) can directly access and use your Web application through these two public links.

### Unpublish

If you need to take the project offline, simply click the AutoCoder Subdomain button at the top of the interface to interrupt external access.

## Liked Template Management

### Likes Panel

Here, all the excellent project templates you have liked in the SparkVibe community will be automatically saved.

<Frame>
  ![Image](/images/image-3.png)
</Frame>

**One-Click Reuse**: You can not only view these inspiration libraries at any time but also directly perform a **Remix** operation on them. With one click, you can convert them into your own new project drafts and continue developing based on the original author's work.
