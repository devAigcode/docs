## What you'll learn

- **Product Surface** — define what you're actually building
- **Context of Use** — clarify who it's for and how it's used
- **Constraints and Taste** — set features, UX direction, and design rules
- **Simple vs optimized prompts** — see the difference side by side using a Wedding Planning Platform

## Why better prompts matter

A simple prompt gives AutoCoder a starting point. An optimized prompt — one that defines the product surface, the context of use, and your constraints and taste — turns the same idea into a full product system. This video uses a Wedding Planning Platform to compare a simple prompt with an optimized prompt, so you can see exactly how structure changes the outcome.
